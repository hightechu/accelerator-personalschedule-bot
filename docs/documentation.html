Personal Schedule Bot

PSB is a user friendly scheduling and reminder bot, to help users stay on top of their tasks and save them free time to do the things they want.

Add to Discord

Commands

!beep

The bot responds with "Boop." with a thumbs up reaction.

!ping

The bot responds with "Pong."

!task [letter]-[time(minutes)]

The bot responds with "Okay, I have added [letter] in the schedule and have assigned it for [time] minutes."

!calender

The bot responds with a schedule for the week based off the tasks set with the command !task.
